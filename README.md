# AIで作ったアプリを無料で公開しよう｜Cloudflareで学ぶ「公開」と「運用」ハンズオン

Claude Code をはじめとしたコーディングエージェントを使うと、アプリが簡単に作れるようになりました。
一方で、作ったアプリを実際にインターネットへ公開する方法や、安全に運用する方法が分からず、そこで
止まってしまう人も少なくありません。

このハンズオンでは、**Cloudflare（クラウドフレア）の無料プラン** を使って、アプリを公開し運用するまでの
流れと、公開する際のセキュリティ上の注意点を、手を動かしながら学びます。

## 対象者

- AI や生成ツールでアプリを作ったことがある方
- 作ったアプリを公開する方法が分からず止まっている方
- Cloudflare を触ったことがない方
- 「無料でどこまでできるのか」を知りたい方
- 個人開発を始めたい方／公開後の運用に不安がある方

## できるようになること

- 作ったアプリをインターネットに公開して友人や自分のスマートフォンからアクセスできるようになる
- インターネット上にWebアプリケーションを公開するときに気をつけておきたいことを学べる

「作る」だけで終わらず、「公開して運用する」ための第一歩を踏み出せます。

## 事前準備

当日スムーズに進めるため、以下を事前にご用意ください。

- **ノートPC** の持参
- Cloudflare アカウントの作成に使う、**メールの送受信ができるメールアドレス**（Google アカウントなど）
- **Node.js** のインストール（[セットアップ手順](./sections/HOST.md)）

Node.js のインストールは事前に行っておいてください。もしわからなければ当日の最初にサポートします。
