# 03-ai-agents - AI エージェントの暴走

「自分で手を動かす AI（AI エージェント）」がなぜ暴走するのかを、最近のニュースを交えて整理する
解説回です（コードは書きません）。曖昧な指示・過剰な権限・幻覚という 3 つの原因を押さえたうえで、
2025 年に起きた Replit の本番 DB 削除、Amazon Q 拡張への破壊コマンド混入、M365 Copilot の
プロンプトインジェクション（EchoLeak / CVE-2025-32711）を取り上げます。

そこから、最小権限・人間の承認・サンドボックス・上限/予算・ログという最低限の守りを学びます。
