🧩 静的ページアプリの例

サーバーがなくても、フロント（ブラウザ）だけでこれくらいは作れます。

下の 3 つは、いずれも HTML と JavaScript だけ で動くアプリです。データを外のサーバーに 送らないので、公開のハードルもセキュリティ上のリスクも低いのが特徴です。気になるものを開いてみましょう。

⚖️ BMI 計算機 身長と体重を入れると、その場で BMI と判定を計算して表示します。
📒 単語帳 1問4択のクイズで単語を覚える。難易度は3段階。ベストは自分のブラウザに保存。
🎹 音のなるおもちゃ ボタンを押すと音がなる、シンプルな鍵盤のおもちゃ。
※ どのアプリも、入力した内容を外のサーバーへ送りません。計算や保存はすべて「あなたのブラウザの中」で 完結します。
