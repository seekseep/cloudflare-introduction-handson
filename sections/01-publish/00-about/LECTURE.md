# このセクションについて

このセクションのゴールはひとつです。

> **とにかく、Web サイトをインターネットに公開する方法を学ぶ。**

手元で作った HTML を、世界中のどこからでもアクセスできる URL に載せる。ここまでを、いちばん短い道のりで
体験します。難しい仕組みの理解は後回しでかまいません。まずは「公開できた」という手応えを掴むことが目的です。

## このセクションの流れ

1. [Cloudflare アカウントを作る](../01-account/LECTURE.md) … 公開に使うアカウントと CLI（wrangler）を準備する
2. [Pages でフロントを公開する](../02-pages/LECTURE.md) … 「ひとことボード」のフロントを Cloudflare Pages に公開する
3. [静的ページアプリの例](../03-examples/LECTURE.md) … サーバーなしで作れるアプリの例を、実際に動かして眺める

## このセクションで分かること

- 作った Web ページを、無料で・サーバー管理なしにインターネットへ公開できること
- 「フロント（見た目）だけ」でも、意外といろいろなアプリが作れること
- サーバーを持たない静的ページは、公開のハードルもセキュリティ上のリスクも低いこと

保存や共有のような「サーバーが必要な処理」は、このセクションの次（[ウェブアプリの基本](../../02-web-app-basic/01-overview/LECTURE.md)
以降）で扱います。まずは「公開」そのものに集中しましょう。
